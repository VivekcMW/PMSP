# Suraj Prakash — Product Manager Portfolio

Personal portfolio for Suraj Prakash, Senior Product Manager (DOOH & Programmatic Ad Tech · AI-Assisted 0→1 Builder).

**Live pages:** Home · Work (+ case study deep dives) · Approach · About · Contact

## Tech Stack

- [Next.js](https://nextjs.org) (App Router) + TypeScript
- Tailwind CSS v4 (light theme, custom tokens in `src/app/globals.css`)
- Framer Motion (reveals, counters, Mission Control hero animations)
- lucide-react icons · Poppins + JetBrains Mono via `next/font`

## Getting Started

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (fully static)
```

## Editing Content

All resume/portfolio content lives in a single file — **`src/data/resume.ts`** (profile, stats, case studies, career timeline, skills, certifications, education). Edit there; no component changes needed.
